# Semantic Layer

`semantic` gives intent-oriented names to system values.

## Rule

Semantic tokens may only reference `system.*`.

## Current Files

- `colors.tokens.json`
- `spacing.tokens.json`

## Current Spike Paths

- `semantic.color.interactive.primary` -> `{system.light.color.brand.primary}`
- `semantic.color.interactive.primary-text` -> `{system.light.color.text.on-brand}`
- `semantic.spacing.component.button.padding` -> `{system.spacing.component.button.padding-horizontal}`
