# System Layer

`system` maps universal primitives into theme and domain level values.

## Rule

System tokens may only reference `universal.*`.

## Current Files

- `theme.colors.tokens.json`
- `theme.spacing.tokens.json`

## Current Spike Paths

- `system.light.color.brand.primary` -> `{universal.color.blue}`
- `system.light.color.text.on-brand` -> `{universal.color.white}`
- `system.spacing.component.button.padding-horizontal` -> `{universal.spacing.16}`
- `system.spacing.component.button.padding-vertical` -> `{universal.spacing.8}`
